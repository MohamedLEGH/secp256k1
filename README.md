secp256k1
=========
Utilities for the secp256k1 curve, as used in Bitcoin. Inspired by https://onyb.gitbook.io/secp256k1-python/.

# Installation
```bash
raco pkg install secp256k1
```

For more info see: https://pkgs.racket-lang.org/package/secp256k1
